# Simple Icons

## Usage

To include the coloured icons simple include the SVG. If youâ€™d like to use colour the SVGs yourself you can add CSS to define `si-solid`, `si-translucent` and `si-transparent` classes. For example to set all icons to be white:

```css
.si-solid {
	fill: white;
}
.si-translucent {
	fill: white;
	opacity: 0.5;
}
.si-transparent {
	opacity: 0;
}
```
